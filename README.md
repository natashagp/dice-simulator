# Dice Simulator Game

### Step 1

Clone the repository

```shell
git clone https://github.com/natashagp/dice-simulator.git
```

### Step 2

Change the directory to dice-simulator

```shell
cd dice-simulator
```

### Step 3

Run project

```shell
python main.py
```
